* 검색 알고리즘
    * 선형 검색 : 배열의 인덱스를 처음부터 끝까지 하나씩 증가시키면서 방문하여 그 값이 속하는지를 검사
    
    * 이진 검색 : 만약 배열이 정렬되어 있다면, 배열 중간 인덱스부터 시작하여 찾고자 하는 값과 비교하며 그보다 작은(작은 값이 저장되어 있는) 인덱스 또는 큰 (큰 값이 저장되어 있는) 인덱스로 이동을 반복

* 알고리즘 표기법
    * Big O
        * O(n^2)
        * O(n log n)
        * O(n) - 선형 검색
        * O(log n) - 이진 검색
        * O(1)
    
    * Big Ω
        * Ω(n^2)
        * Ω(n log n)
        * Ω(n) - 배열 안에 존재하는 값의 개수 세기
        * Ω(log n)
        * Ω(1) - 선형 검색, 이진 검색
