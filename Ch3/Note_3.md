* 컴파일링
    * 전처리 : 실질적인 컴파일이 이루어지기 전에 무언가를 실행
    
    * 컴파일링 : 소스 코드에서 오브젝트 코드로 변환하는 전체 과정을 통틀어 일컫기도 하지만, 구체적으로 전처리한 소스 코드를 어셈블리 코드로 변환시키는 단계
    
    * 어셈블링 : 어셈블리 코드를 오브젝트 코드로 변환
    
    * 링킹 :  여러 개의 파일로 이루어져 있어 하나의 오브젝트 파일로 합쳐져야 한다면 링크라는 컴파일의 마지막 단계
